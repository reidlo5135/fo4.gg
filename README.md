## **FO4.GG?**
<b>FO4.GG</b>란 Nexon Developers에서 제공하는 fifaonline4 API를 이용하여 사용자의 검색을 통하여 fifaonline4 사용자 정보를 제공해주는 사이트입니다. <br><br>

## <b>주요기능</b>

### [서비스 주요 기능]
- <b>fifaonline4 사용자(구단주) 정보 검색</b> : 사용자가 닉네임을 통하여 검색 시 구단주 정보 제공

### [제공 정보]
- <b>구단주 닉네임</b>
- <b>구단주 레벨</b>
- <b>구단주 역대 최고 공식경기 레이팅</b>
